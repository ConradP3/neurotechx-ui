<!-- INSTALLATION -->
### Installation
For MacOs:
```
cd api
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cd ..
python api/api.py runserver -d -r
```
In Second Terminal:
```
npm install
npm start
```
